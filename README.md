# AI Crop Advisory System | Bhartiya Antariksh Hackathon 2024

**Team 100XX** - Thapar Institute of Engineering and Technology

An **AI-powered solution** for automated crop type mapping, phenology-aware moisture stress detection, and intelligent irrigation advisories using fused Optical and Microwave satellite remote sensing data.

---

## 🎯 Project Objective

Develop an operational system that:
- Automatically identifies crop types using multi-temporal satellite data
- Detects moisture stress across different growth stages
- Generates 8-day crop water deficit maps and actionable irrigation advisories
- Supports precision agriculture and canal command area water management

---

## 🚀 Key Features

- **Multi-source Data Fusion**: Optical (Sentinel-2, LISS-III, AWiFS, MODIS) + Microwave SAR (Sentinel-1, EOS-04)
- **Phenology-Aware Analysis**: Growth stage specific stress detection
- **Irrigation Advisory Engine**: Weekly (8-day) water deficit estimation
- **Scalable Architecture**: Designed for Google Earth Engine + Cloud deployment
- **All-weather Monitoring**: Reliable performance during monsoon using SAR data
